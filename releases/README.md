# Releases

每个版本一个文件夹,内含可直接复制覆盖的 `module.zip` + 上传给 Foundry 的 `module.json` + 本版本 `NOTES.md`。

线下分发补丁时,把对应 `vX.Y.Z/` 目录的 `module.zip` 给对方,让对方解压到自己的 `Data/modules/local-asset-library/` 即可。
